# Music-Bot

# DESCRIPTION
- A Python Music Discord Bot capable of playing songs and playlists from youtube.

# INSTALLATION REQUIREMENTS
  -PYNACL
  -PYCORD
  -YT-DLP
  -GOOGLE-API-PYTHON-CLIENT

# COMMAND OPTIONS
  -Slash Commands
  -/searchsong     search song by title and queues into playlist
  -/clear          clears playlist and resets music bot state
  -/current        sends embed of information about current song
  -/dc             disconnects bot from the voice channel
  -/goto           skips to/plays song by index in the playlist
  -/help           sends embed of all command options the music bot has to offer
  -/loop           loops current song
  -/list           send embed of music playlist/songs by index 
  -/pause          pauses current song
  -/previous       plays previous song
  -/queue          queue a song or a playlist by youtube url to playlist
  -/resume         resumes current song
  -/shutdownbot    disconnects bot from voice channel and shuts down program
  -/skip           skips to next song
